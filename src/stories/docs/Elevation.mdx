# Elevation

Quill shadows use a warm sepia formula — ink at low alpha — so they feel like
cast ink, not grey box-shadows. A negative spread keeps the shadow tight under
the element. Values render live from the token source (`tokens.shadow`, Dawn
recipes shown; each theme carries its own casts).
