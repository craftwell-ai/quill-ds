# Spacing

Quill uses a 4px base grid, with half-steps and control sizes filling the gaps
components actually need. Values render live from the token source
(`tokens.spacing`, rem × 16).

{Object.entries(tokens.spacing)
  .sort(([a], [b]) => parseFloat(a) - parseFloat(b))
  .map(([step, rem]) => (
    <SpaceRule key={step} token={`space-${step}`} px={parseFloat(rem) * 16} usage={USAGE[step] ?? ''} />
  ))}

## Usage guide

| Scale | Token | Use |
|---|---|---|
| 2–8px | `space-0.5` – `space-2` | Icon gaps, badge insets, hairline offsets |
| 10–16px | `space-2.5` – `space-4` | Component internals, input padding, small card spacing |
| 20–24px | `space-5`, `space-6` | Form spacing, card padding, list row gaps |
| 28–40px | `space-7` – `space-10` | Control heights, section padding, breathing room |
| 48–96px | `space-12` – `space-24` | Hero insets, page-level padding, section separators |
